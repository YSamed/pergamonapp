# Core Features

## Habit sistemi
- Günlük / haftalık / aylık habit oluşturma
- Belirli günlerde tekrar eden habitler
- Completion tracking
- Streak sistemi
- Habit difficulty / XP değeri
- İsteğe bağlı reminder sistemi

## Todo sistemi
- One-off task oluşturma
- Deadline / due date
- Priority
- Estimated effort
- Quick add
- Bugün, yaklaşanlar, gecikenler görünümleri

## XP ve seviye sistemi
- Her görev/habit bir XP değeri üretir
- XP; kullanıcı level’ını artırır
- Ayrıca belirli skill alanlarına XP dağıtılabilir
- Örn: Fitness, Career, Learning, Mindset, Social
- Kullanıcı genel level + kategori bazlı skill level görebilir

## Skill sistemi
- Kullanıcı kendi gelişim alanlarını seçebilir veya oluşturabilir
- Örnek skill’ler:
  - Strength
  - Focus
  - Discipline
  - Coding
  - Communication
  - Health
- Görevler bir veya birden fazla skill’e bağlanabilir

## Streak ve consistency sistemi
- Günlük devamlılık takibi
- Habit özel streak
- Genel aktif gün streak’i
- Haftalık consistency score

## Achievement / badge sistemi
- İlk 7 gün streak
- 100 task complete
- 30 gün habit tamamlandı
- Clan contribution başarısı
- Boss week completion gibi event bazlı achievement’lar

## Social features
- Arkadaş ekleme / davet
- Aktivite akışı
- React / support / cheer mekanikleri
- Friendly competition
- Accountability nudges

## Challenge / event sistemi
- Günlük mini challenge’lar
- Haftalık clan challenge’ları
- Sezonluk eventler
- Tematik progression eventleri

## Avatar / profil sistemi
- Kullanıcı seviyesi ve görünür progression
- Basit avatar customization
- Rozetler
- Clan rozetleri
- Profil vitrini

## MVP kapsamı
İlk versiyon için önerilen MVP:
- Auth
- Onboarding
- Habit ekleme / düzenleme / tamamlama
- Todo ekleme / düzenleme / tamamlama
- XP kazanma
- Level sistemi
- Basit skill mapping
- Streak sistemi
- Basit profil ekranı
- Basit clan sistemi
- Basit activity feed

## Roadmap fikirleri
- Gelişmiş avatar editor
- Marketplace / itemization
- Deep analytics
- AI coach
- Voice input
- Widget’lar
- Smart reminder automation
- Advanced clan wars
