# Gamification

## Temel yaklaşım
Bu uygulama için gamification katmanı kozmetik değil, core product’tır.

## Ödül döngüsü
1. Kullanıcı görev ekler
2. Görevi tamamlar
3. Hemen görsel/duygusal ödül alır
4. XP artar
5. Skill veya level progress görünür
6. Streak / achievement / clan contribution tetiklenebilir
7. Kullanıcı tekrar aksiyon almak ister

## Dikkat edilmesi gerekenler
- XP inflation olmamalı
- Her şey XP verirse sistem anlamsızlaşır
- Zor görevler daha değerli olabilir
- Günlük spam task açıp sistemi kırma önlenmeli
- Habit completion ve todo completion farklı ağırlıklandırılabilir

## Olası XP mantığı
Örnek:
- Easy task: 5 XP
- Medium task: 10 XP
- Hard task: 20 XP
- Daily habit complete: 5-15 XP
- Streak milestone bonus: ekstra XP
- Clan event completion: bonus XP

## Progression tasarım hedefi
- Kullanıcı “bir şey işaretledim” değil, “ilerledim” hissi yaşamalı
- Progress görünürlüğü sürekli ve anlaşılır olmalı
- Küçük görevler küçük ödül, zor görevler daha anlamlı ödül üretmeli
- Sistem exploit’e açık olmamalı
