# UX and IA

## Onboarding yaklaşımı
Onboarding uzun ve sıkıcı olmamalı.

Amaç:
- Kullanıcının ilk 2 dakika içinde ilk win’i yaşaması

Önerilen akış:
1. Hedef seçimi
2. 3 odak alanı seç
3. İlk habit ekle
4. İlk todo ekle
5. İlk skill eşleşmesini yap
6. İlk completion deneyimi
7. XP kazanımını göster
8. İsteğe bağlı clan join öner

## UX yönlendirmeleri
Claude Code ürün geliştirirken şu UX ilkelerini korusun:
- Ana ekran karmaşık olmamalı
- Günün en önemli aksiyonları öne çıkmalı
- Completion etkileşimi tatmin edici olmalı
- Progress bar’lar motive edici olmalı
- Gamification çocukça değil, clean ve modern hissettirmeli
- Sosyal alanlar ana işlevi bozmamalı
- Kullanıcıya sürekli “sen gelişiyorsun” hissi verilmeli

UI tonality:
- Modern
- Clean
- Slightly premium
- Motivating
- Energetic but not childish

## Bilgi mimarisi
Önerilen ana navigation tab’leri:
- Home
- Tasks
- Progress
- Clan
- Profile

### Home
- Today plan
- Habits due today
- Priority todos
- Daily XP summary
- Quick complete / quick add

### Tasks
- Habit list
- Todo list
- Filter / sort
- Add new

### Progress
- Total XP
- Level
- Skill bars
- Streaks
- Achievements
- Weekly stats

### Clan
- Clan overview
- Member activity
- Weekly contribution
- Clan challenges
- Join / invite

### Profile
- Avatar
- Badges
- Preferences
- Reminder settings
- Account
