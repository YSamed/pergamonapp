# Metrics and Monetization

## Başarı metrikleri
### Aktivasyon
- İlk gün ilk habit oluşturma oranı
- İlk gün ilk task completion oranı
- Onboarding completion rate
- İlk XP kazanımına ulaşma süresi

### Engagement
- DAU / WAU
- Günlük tamamlanan task sayısı
- Haftalık aktif gün sayısı
- Ortalama streak uzunluğu
- Kullanıcı başına weekly XP event sayısı

### Sosyal kullanım
- Clan join rate
- Clan içi haftalık etkileşim
- Activity feed interaction rate
- Challenge katılım oranı

### Retention
- D1 / D7 / D30 retention
- 2. hafta streak devam oranı
- Clan üyelerinde retention farkı

## Monetizasyon fikirleri
Muhtemel monetizasyon modeli:
- Freemium
- Premium subscription
- Yıllık plan odaklı teklif
- Lifetime opsiyonu olabilir

Premium alanlar:
- Gelişmiş analytics
- Daha fazla customization
- Sınırsız skill / challenge / clan tools
- Premium avatar cosmetics
- Gelişmiş progression raporları
- Özel sezon içerikleri
